# About DV Debug Lab

DV Debug Lab is a practical knowledge base for design verification engineers.

The goal is simple:

> Turn painful verification-debug problems into clear articles, examples, and checklists.

This site focuses on:

- SystemVerilog
- UVM
- gate-level simulation
- CDC
- X-propagation
- assertions
- functional coverage
- regression scripting
- interview preparation

## Philosophy

Most verification resources explain concepts. DV Debug Lab explains what to do when things break.

Because things will break. Silicon and simulators both have a flair for drama.
